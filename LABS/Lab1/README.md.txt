Insturctions for completeing Lab 1
Use the index.html template to create your first website.
Set up the class technologies as explained in the The Tools of Internet Publishing video.
Github, Github Desktop, Notepad++ (or similar code editor).
Create a new Github repository called webdev_lab1.
Remember to "Initialize this repository with:" by checking "Add a README file" on the repository setup page.
Follow the instructions in the The Tools of Internet Publishing video for linking your new repository with a folder on your computer, using Github Desktop.
Cut and paste the index.html file from the lab 1 repository into your webdev_lab1 repository.
Add three <p> elements to index.html with some basic information about yourself.
Follow the instructions in the Lesson 2: The Tools of Internet Publishing video for creating a Github Pages live link.
Finally, cut and paste the webdev_lab1 Repository URL (Repo Link) and Github Pages URL (Live Link) into this assignment's turn-in post. Like so:
Repo Link: https://github.com/dkriegls/coursera/tree/main

Live Link: https://dkriegls.github.io/coursera/
